# ClipBuilder Frontend (React + Vite + Tailwind)

## Pré-requisito
- Node.js 18+ e npm.

## Rodar localmente

```bash
npm install
npm run dev
```

- Frontend: http://localhost:5173
- Backend esperado em: http://127.0.0.1:8000

## Exportação

O formato de saída é configurável nas Configurações (modal). Suporta `markdown` (ZIP), `html`, `docx`, `pdf` e `plain`.
